Odoo Delivery Boy

Assign and Manage your delivery of orders with delivery boys

Available in Android and IOS platform

Odoo delivery boy app provides easy user interface for delivery boys.

Makes it easy for delivery boys to track and manage their orders.

Provides GPS tracking feature for easy and fast delivery.

Can track the commission earned for each delivery.

Provide multilingual feature for delivery boys to understand app better.

This Module Allows

  • Full Compatibility with Mobikul
  • Real-time assignment of orders to Delivery Boys.
  • Native mobile app for delivery boys to manage and track orders delivery.
  • Management of commission of delivery boys for order deliveries.
  • Real-time order status update through native mobile app.

  • Full Admin Control
    • Auto validate the orders delivery manually and automaticaly
    • Auto generate commission invoice when order is delivered manually and automaticaly
    • Set commission amount provided to a delivery boy when an order is delivered
    • Configure delivery boy app
  • Admin View
  • Configuration
